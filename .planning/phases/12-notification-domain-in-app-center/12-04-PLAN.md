  Mount (in 12-05): `app.use('/api/notifications', verifyIdToken, notificationRouter)` — verifyIdToken ONLY, NOT requireAdmin. Inside the router: `const uid = req.auth.uid;` NEVER req.body.uid / req.params.uid (V4 IDOR).

  Endpoints:
    GET  /                         → cursor feed { items, nextCursor }; filter { uid: req.auth.uid }
    GET  /unread-count             → { count } where { uid, read:false }
    PATCH /:id/read                → mark one read; filter { _id, uid } (ownership via uid, not _id alone)
    PATCH /read-all                → mark all read for { uid }
    POST /subscriptions            → create (schemas.js discriminatedUnion); attach uid from token; watch → events all 4; cadence default instant
    GET  /subscriptions            → list { uid, active:true }
    PATCH /subscriptions/:id       → edit cadence (saved_search) / events (watch); filter { _id, uid }
    DELETE /subscriptions/:id      → soft or hard delete; filter { _id, uid }

  Cursor: encodeCursor/decodeCursor COPIED VERBATIM from moderation/router.js:27-45 (S1). Query shape:
    if (cursor) query.$or = [ {createdAt:{$lt:cursor.createdAt}}, {createdAt:cursor.createdAt,_id:{$lt:cursor._id}} ];
    rows = Notification.find({uid, ...query}).sort({createdAt:-1,_id:-1}).limit(limit+1).lean();
    limit clamp: Math.min(Math.max(rawLimit,1),100), default 25.
  Bad cursor → 400 invalid_cursor (decodeCursor returns undefined sentinel).

  Error table: KNOWN_USER_ERRORS set + handleServiceError(err,res,tag) shape from moderation/router.js:59-84 (S4): subscription_not_found, invalid_cursor, not_owner, invalid_payload; unknown → 500 internal_error.
  Per-route schema gate: safeParse → 400 invalid_payload (router.js:96-99 shape).
