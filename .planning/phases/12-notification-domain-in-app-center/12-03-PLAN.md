<interfaces>
emit(event) event shapes (RESEARCH §Pattern 3, PATTERNS §emit trigger points):
  { type: 'new_listing', carId, actorUid }
  { type: 'price_drop', carId, actorUid, oldPrice, newPrice }   // emit only if newPrice < oldPrice
  { type: 'booked', carId, actorUid }
  { type: 'sold', carId, actorUid }
  { type: 'back_available', carId, actorUid }   // only when old listingStatus 'booked' → new 'active'

Guard sequence inside emit (NDOM-03):
  (a) hide-hook: `const visible = await Car.findById(carId)` — PLAIN, NO setOptions/includeAllUsers/includeAllListingStatuses. `if (!visible || visible.status !== 'active') return;` (TOCTOU re-check; PATTERNS §Guard a; Car.js:64-121).
  (b) actor-exclusion: drop any subscription where `subscription.uid === event.actorUid`.
  (c) dedup: dedupeKey = `${carId}:${eventType}`; per (uid, carId, eventType) produce ≤1 notification (check existing row / unique-ish guard within window).

Target resolution:
  new_listing → matchSavedSearches(car) → matching saved_search subscriptions.
  watch events → Subscription.find({ kind:'watch', carId, active:true }), filtered by events array containing the eventType.

matchSavedSearches(car): pure module (capabilities.js analog). Query Subscription {kind:'saved_search', active:true, 'criteria.makeId', 'criteria.modelId'} then numeric-bound (priceMin/Max, yearMin/Max) + bodyType filtering. criteria.makeId/modelId are ObjectId — compare against car.makeId/modelId (ObjectId). Off the hot path (Pitfall 8).

fcm.js: `module.exports = { send: async () => ({ ok: true, delivered: 0, stub: true }) }` (RESEARCH A4, NPRF-07).

translations.js: RU-first map keyed by titleKey/bodyKey interpolating params; currency formatted as KGS som (NI18N-03). Keys at minimum: new_match, price_drop, booked, sold, back_available (title + body each).

schemas.js: zod (S3) — kindEnum ['saved_search','watch'], cadenceEnum ['instant','daily'], eventEnum ['price_drop','booked','sold','back_available'], criteriaSchema (.strict, makeId/modelId/priceMin/priceMax/yearMin/yearMax/bodyType all optional), discriminatedUnion('kind',...) for the create payload.
</interfaces>
