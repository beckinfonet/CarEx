|----------|-------------|
| client → PUT/PATCH /api/cars | No auth middleware; actor identified via req.body.sellerId (ownership IS enforced: car.sellerId !== sellerId → 403) |
| client → /api/notifications | verifyIdToken; uid from token |
| client → PUT /api/users/:uid | language enum-validated in the whitelist |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-05-01 | Tampering | actor-exclusion source | mitigate | Actor = ownership-enforced req.body.sellerId / buyerUid / adminUid; cannot be spoofed to someone else's car (ownership 403). Do NOT expand auth scope (RESEARCH OQ#3, no-breaking-changes constraint) |
| T-12-05-02 | Info disclosure | emit on hidden listing | mitigate | emit re-reads Car with plain findById (handled in 12-03); call site passes only carId/actor, never a cached visible Car |
| T-12-05-03 | DoS | emit on hot path | mitigate | emit wrapped so failure never breaks the listing/booking response; matching is off-hot-path (Pitfall 8) |
| T-12-05-04 | Tampering | PUT /api/users language | mitigate | enum ['RU','EN'] guard in the whitelist (NI18N-01) |
</threat_model>
