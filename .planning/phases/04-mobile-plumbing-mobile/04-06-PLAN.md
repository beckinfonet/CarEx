| T-04-06-05 | Elevation of Privilege | component re-renders cause useAppStateRefresh to re-register listener with stale closure | accept | Hook's useEffect dep array includes `refresh`, so re-registration is intentional and correct — each re-subscription uses the latest callback. Not a vulnerability. |
