<objective>
Migrate every backend-facing call in `src/services/AuthService.ts` from plain `axios.X(\`${API_URL}/...\`)` to the shared `apiClient.X('/...')`. Critical invariant: ZERO new methods added to AuthService.ts during this milestone (MOB-01). Migration is surgical — swap the axios target per method, nothing else. The module remains structurally identical; only the HTTP client underneath changes.

Purpose: MOB-02 (shared apiClient used by both AuthService and ModerationService) and MOB-01 (AuthService no-new-methods guardrail maintained).

Output: Modified `src/services/AuthService.ts`. No new files.
