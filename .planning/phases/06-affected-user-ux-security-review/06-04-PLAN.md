```
gate{Capability}{Severity}{Title|Body}
  where Capability ∈ { CreateListing, CreateOrder, ApplyProvider, ContactSeller }

```

```typescript
const SEVERITY_TO_KEY_PART = {
  feature_limited: 'FeatureLimited',
  blocked_with_review: 'Blocked',
  permanently_banned: 'Banned',
} as const;
```

Capability-key naming (PascalCase for key assembly):
```typescript
const CAPABILITY_TO_KEY_PART = {
  create_listing: 'CreateListing',
  create_order: 'CreateOrder',
  apply_as_provider: 'ApplyProvider',
  contact_seller: 'ContactSeller',
} as const;
```

Example assembly: `capability='create_listing'`, `state='feature_limited'` → key prefix `gateCreateListingFeatureLimited` → `${prefix}Title` and `${prefix}Body`.
