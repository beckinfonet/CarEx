| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-01-04 | Denial of service | `{ sellerId: 1, status: 1 }` index creation | accept | Index is small (sellerId + 4-value status); MongoDB Atlas index-build is online-by-default; no migration-time DoS risk for the existing Car collection size (small-to-mid CarEx production estimate per [[backend_deploy_gotcha]]). |
